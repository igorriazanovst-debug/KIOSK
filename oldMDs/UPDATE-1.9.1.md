# 🔄 Обновление v1.9.1 - Улучшенное расположение панели структуры

## Изменение

Панель структуры виджетов теперь находится **под** библиотекой виджетов в одной колонке, а не как отдельная панель.

## Расположение

### Было (v1.9.0):
```
┌─────────────┬─────────────┬────────────┬──────────────┐
│  Библиотека │  Структура  │   Холст    │  Свойства    │
│  виджетов   │             │            │              │
│   220px     │   220px     │    flex    │    300px     │
└─────────────┴─────────────┴────────────┴──────────────┘
```

### Стало (v1.9.1):
```
┌─────────────┬───────────────────────────┬──────────────┐
│  Виджеты    │                           │              │
│  [список]   │                           │              │
│─────────────│         Холст             │  Свойства    │
│  Структура  │                           │              │
│  [список]   │                           │              │
│   220px     │          flex             │    300px     │
└─────────────┴───────────────────────────┴──────────────┘
```

## Преимущества

✅ **Компактнее** - больше места для холста  
✅ **Логичнее** - связанные панели вместе  
✅ **Удобнее** - добавил виджет → сразу видно в структуре

## Технические детали

**Изменённые файлы:**

1. **WidgetLibrary.tsx**
   - Добавлен импорт OutlinePanel
   - OutlinePanel добавлена в конец структуры
   ```tsx
   <OutlinePanel />
   ```

2. **WidgetLibrary.css**
   - `height: 100%` и `overflow: hidden` для контейнера
   - `flex-shrink: 0` для header, content и footer
   - Content больше не `flex: 1`

3. **OutlinePanel.css**
   - `width: 100%` вместо `220px`
   - `border-top` вместо `border-right`
   - `flex: 1` для заполнения оставшегося места
   - `min-height: 0` для корректной работы scroll

4. **Editor.tsx**
   - Убран `<OutlinePanel />` из основного layout
   - OutlinePanel теперь внутри WidgetLibrary

## Как это работает

**Flex-контейнер:**
```css
.widget-library {
  display: flex;
  flex-direction: column;
  height: 100%;
}
```

**Секции:**
- Header: `flex-shrink: 0` (фиксированный)
- Content: `flex-shrink: 0` (по содержимому + scroll)
- Footer: `flex-shrink: 0` (фиксированный)
- OutlinePanel: `flex: 1` (заполняет остаток)

**Прокрутка:**
- Widget content: `overflow-y: auto`
- Outline list: `overflow-y: auto`
