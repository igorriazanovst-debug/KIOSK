# Быстрый старт

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd kiosk-content-platform
```

2. Установите зависимости:
```bash
npm install
```

## Запуск редактора

```bash
npm run dev:editor
```

Редактор откроется в браузере по адресу `http://localhost:3000`

## Использование редактора

### Создание проекта

1. При первом запуске автоматически создается новый проект с разрешением 1920x1080
2. Измените размеры холста в настройках проекта при необходимости

### Добавление виджетов

1. В левой панели "Виджеты" выберите нужный тип
2. Кликните по виджету - он добавится на холст
3. Перетащите виджет в нужное место
4. Измените размер с помощью угловых маркеров

### Редактирование свойств

1. Выберите виджет на холсте
2. В правой панели "Свойства" измените параметры
3. Доступны настройки:
   - Позиция (X, Y)
   - Размер (Ширина, Высота)
   - Поворот
   - Специфичные свойства виджета

### Горячие клавиши

- `Ctrl/Cmd + Z` - Отменить
- `Ctrl/Cmd + Shift + Z` - Повторить
- `Ctrl/Cmd + C` - Копировать
- `Ctrl/Cmd + V` - Вставить
- `Ctrl/Cmd + X` - Вырезать
- `Delete` / `Backspace` - Удалить выделенные виджеты
- `Ctrl/Cmd + Click` - Множественное выделение

### Сохранение и загрузка

1. Нажмите кнопку "Сохранить" на панели инструментов
2. Файл сохранится с расширением `.kiosk.json`
3. Для загрузки нажмите "Открыть" и выберите файл

### Настройки отображения

- **Сетка** - включение/выключение направляющей сетки
- **Привязка к сетке** - автоматическое выравнивание по сетке
- **Zoom** - изменение масштаба холста (10% - 300%)

## Типы виджетов

### Прямоугольник
Базовый геометрический элемент

### Кнопка
Интерактивная кнопка с текстом
- Настройки: текст, цвет фона, цвет текста

### Текст
Текстовый блок
- Настройки: текст, размер шрифта, цвет

### Изображение
Отображение изображения
- Настройки: URL изображения

### Видео
Видеоплеер
- Настройки: URL видео, автовоспроизведение, повтор

## Формат файла проекта

Проекты сохраняются в JSON-формате:

```json
{
  "version": "1.0.0",
  "name": "Новый проект",
  "canvas": {
    "width": 1920,
    "height": 1080,
    "backgroundColor": "#ffffff"
  },
  "widgets": [
    {
      "id": "widget-123",
      "type": "button",
      "x": 100,
      "y": 100,
      "width": 200,
      "height": 80,
      "properties": {
        "text": "Кнопка",
        "backgroundColor": "#4a90e2"
      }
    }
  ]
}
```

Если у вас возникли проблемы:
1. Проверьте консоль браузера на наличие ошибок
2. Убедитесь, что все зависимости установлены
3. Попробуйте очистить кэш браузера
4. Создайте issue в репозитории проекта
